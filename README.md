# TutorialMod
 Mod for tModLoader(Terraria1.4)

## アイコンについて
### icon
iconはModリストで表示されるアイコンです。ワークショップにアップロードした際にサムネイルとしても表示されます。
80*80の画像が推奨されます
### icon_small
icon_smallはbestiaryにどのModのNPCなのかという情報として表示される、小さいアイコンです。
30*30の画像が推奨されます

## .editorconfig
editorconfigはプロジェクトでのコードスタイルや書き方とかを統一化してくれるすごい奴
このプロジェクトでは保存するファイルの文字コードの変更が主な役割。
フォント入れてるのに日本語を使おうとすると文字化けするって方はコピーして使いましょう。
VS2022以前は拡張機能をインストールする必要がありました。
